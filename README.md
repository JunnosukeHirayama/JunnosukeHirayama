## Projects
### [ロウソクの炎の2次元数値解析](https://github.com/JunnosukeHirayama/CandleShapeAnalysis)
- **概要:** ロウソクの炎が形成する特徴的な「ティアドロップ型」の形状と、周囲の空気の対流（流速場）をシミュレーションするPythonプログラムです。外部の重い流体解析（CFD）ライブラリを使用せず、物理現象を支配する偏微分方程式をNumPyを用いた配列演算のみでスクラッチ実装しています。

### [SEO業務における記事内部リンクの管理効率化WEBアプリ](https://github.com/JunnosukeHirayama/GAS-app.1)
- **概要:** WEBメディアのSEO業務における記事の内部リンク管理を効率化するためのアプリです。具体的にはどの記事がどの記事へ内部リンクが繋がっているのかや、 記事ごとの保有リンクの数、SEOで重要になるトピッククラスターの管理を視覚的に分かりやすくし、内部リンク情報の追加削除も直感的に行えるようにするためのものです。

### [2Dランゲーム](https://github.com/JunnosukeHirayama/RunGame)
- **概要:**「スーパーマリオ」のようなクラシックな2Dアクションゲームの操作感にインスパイアされ、Unityでゼロから開発した横スクロールアクションゲームです。 単なるジャンプアクションにとどまらず、手元に戻ってくる「ブーメラン」による遠距離攻撃や、HP・無敵時間を持つボス戦など、プレイヤーを飽きさせない拡張要素を実装しています。
